## Competencias Requeridas

### Obligatorias

1. **TypeScript** - Dominio sólido del lenguaje
2. **React** - hooks, componentes, renderizado
3. **Next.js** - App Router, Server Components, API Routes
4. **Tailwind CSS** - Diseño de interfaces responsivas
5. **Autenticación** - NextAuth.js y seguridad

### Importantes

7. **Zod** - Validación de datos
8. **React Hook Form** - Manejo de formularios
9. **SWR o TansTack Query** - Gestión de estado de servidor
10. **Radix UI / shadcn/ui** - Componentes accesibles

### Opcionales / Avanzadas

11. **PDF.js** - Manipulación de documentos
12. **Chart.js / Recharts** - Visualización de datos
13. **Prisma Adapter para NextAuth** - Gestión de sesiones
14. **Tailwind 4** - Nuevas features de CSS

### Conceptos Clave

- Server-Side Rendering (SSR)
- Static Site Generation (SSG)
- Client Components vs Server Components
- Middleware de Next.js
- ORM y modelado de bases de datos
- REST APIs con Route Handlers
- Manejo de estado global vs local
- Autenticación basada en sesiones/JWT
